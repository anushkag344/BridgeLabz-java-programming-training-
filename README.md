# Object Oriented Programming (OOP) - Basics

This project contains Java programs developed to understand and implement the fundamental concepts of Object-Oriented Programming (OOP).

## Programs Included

### 1. Student.java
A basic program demonstrating the creation of a Student class with attributes and methods.

### 2. Employee.java
A program to manage employee details using classes and objects.

### 3. Circle.java
Calculates the area and circumference of a circle using class methods.

### 4. Ball.java
Demonstrates object creation and manipulation using a Ball class.

### 5. Line.java
Implements the Line Comparison Problem by calculating and comparing line lengths.

### 6. LibraryManagementSystem.java
A simple Library Management System demonstrating OOP concepts such as:
- Classes and Objects
- Encapsulation
- Methods
- Data Management

## OOP Concepts Covered

- Classes and Objects
- Constructors
- Encapsulation
- Abstraction
- Method Creation
- Object Interaction
- Data Hiding
- Code Reusability

## How to Run

Compile the program:

```bash
javac FileName.java
```

Run the program:

```bash
java FileName
```

Example:

```bash
javac Student.java
java Student
```
