# Makerplan Project

This folder contains basic Java programs developed as part of the BridgeLabz Core Programming training.

## Programs Included

### 1. Factorial.java
Calculates the factorial of a given number using Java.

**Example:**
Input: 5

Output: 120

---

### 2. Primeno.java
Checks whether a number is prime or not.

**Example:**
Input: 13

Output: Prime Number

---

### 3. SnakeLadder.java
Simulates the Snake and Ladder game using Java programming concepts.

Features:
- Dice rolling
- Snake movement
- Ladder movement
- Position tracking
- Winning condition

---

## Technologies Used

- Java
- Object-Oriented Programming Basics
- Control Flow Statements
- Loops and Conditional Statements

## How to Run

Compile the Java file:

```bash
javac FileName.java
```

Run the program:

```bash
java FileName
```

Example:

```bash
javac Factorial.java
java Factorial
```

- Understanding Java syntax
- Working with loops and conditions
- Problem-solving using programming
- Implementing simple game logic
